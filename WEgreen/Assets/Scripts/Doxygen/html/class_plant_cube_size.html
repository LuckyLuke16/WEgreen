Detailed Description

Used to create the visible cube element of the AR measuring function.

The cube already exists within the MeasurePrefab of each plant model. The final mesh created in MeasurePlant is used to set the bounds of the cube and display it to the user. Due to variations in scaling, different plant models use different methods within the class.
