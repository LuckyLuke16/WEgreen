Member Function Documentation

◆ addPlant()

void AR_Cursor.addPlant ( )
private

Selected plant models is set on current cursor position.

A maximum of three plants can be placed. The plant gameobjects are added to the placedPlants array. If the amount of placed plants exceeds three a dialoug is set active, that notifies the user, that he has to delete the plants in order to place new ones.

◆ updateCursorAndPlant()

void AR_Cursor.updateCursorAndPlant ( )
private

Updates the position of the cursor, if a hit with a plane occured.

The cursor position is set to the middle of the screen.The cursor as well as the active plant gameobject are rendered to the position of the last hit of the raycastmanager with the detected plane in the middle of the screen.
